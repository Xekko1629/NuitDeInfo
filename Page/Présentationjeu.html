CUBEMAN

CubeMan est une démo d'un plateformer 3D crée par Luca.

Le jeu a été développé grâce à deux API (Application programming interface): WebGl, et Three.js.

Utilisez Z et S pour avancer et reculer.

Utilisez Q et D pour tourner.

Utilisez les flèches directionnelles pour tourner la caméra.

Utilisez la barre espace pour sauter.
